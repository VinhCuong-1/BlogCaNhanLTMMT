# Thư Mục Hình Ảnh

Đặt các file hình ảnh vào thư mục này.

## Chứng Chỉ (Certificates)

Đặt các file chứng chỉ với tên sau:

1. **networking-basics.jpg** (hoặc .png) - Chứng chỉ Networking Basics
2. **javascript-essentials-1.jpg** (hoặc .png) - Chứng chỉ JavaScript Essentials 1
3. **javascript-essentials-2.jpg** (hoặc .png) - Chứng chỉ JavaScript Essentials 2

## Avatar

- **avatar.jpg** (hoặc .png) - Ảnh đại diện của bạn

## Đồ Án (Projects)

- **project-1.jpg** - Hình ảnh đồ án 1
- **project-2.jpg** - Hình ảnh đồ án 2
- ... (thêm tùy ý)

## Lưu Ý

- Tên file không dấu, dùng dấu gạch ngang `-` thay vì khoảng trắng
- Hỗ trợ định dạng: .jpg, .jpeg, .png, .webp
- Nên nén hình ảnh trước khi upload để tăng tốc độ tải
